About the Project

Brief Description

Discover BC is a calm, photography-led travel booking experience designed to reduce cognitive load and support users in making stress-free travel decisions. The experience centres around minimal UI, clean spacing, and emotional clarity.

Creative Problem

Many travel apps create cognitive overload with dense UI, multiple competing actions, and overwhelming information. The challenge was to design a travel experience that feels calm, emotionally grounding, and intuitive for first-time or anxious travelers.

Solution

The interface uses soft visual hierarchy, large photographic anchors, and simplified navigation to create a peaceful digital environment. The design helps users make decisions at their own pace without feeling overwhelmed.

Process & Development

A calm, structured process shaped the final UI screens.

Wireframes

Early wireframes focused on reducing clutter, spacing content generously, and letting imagery guide navigation.

Visual Style Exploration

The visual identity emphasizes soft gradients, clean typography, and tone-on-tone UI components for a quiet, expansive feel.

Final UI Screens

The final UI promotes clarity and emotional ease through gentle composition.
