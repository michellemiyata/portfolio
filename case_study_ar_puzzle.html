About the Filter

Concept

Extending the emotional narrative of the "Puzzle Piece" project into a shared social experience. This TikTok AR filter uses facial tracking to detect when a user smiles, triggering the appearance of the "missing piece" to complete the visual.

Interactive Logic

The effect relies on a custom interaction patch that monitors the mouth curvature. When the "smile" value crosses a threshold (0.7), the opacity of the puzzle graphic interpolates from 0% to 100%, creating a gentle, rewarding feedback loop.
