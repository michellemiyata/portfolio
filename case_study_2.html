About the Production

Brief Description

This project reimagines the TTC Line 1 commute as an emotional timeline rather than a physical map. Each station is represented by colour-coded moods that reflect the inner experience of commuting.

Creative Problem

Traditional transit maps show routes but not emotional experience. The challenge was to interpret a public system through a human, emotional lens.

Solution

Using a Vignelli-inspired colour system, each TTC station becomes an emotion. The final visual reads like an emotional map of the rider’s inner journey.

Process & Development

Exploration included emotional research, colour theory, and mapping emotional pacing across stations using Miro.

Emotional Colour Framework

  • Yellow — Relief / Lightness
  • Blue — Heaviness / Sadness
  • Red — Anger / Overwhelm
  • Black — Fear / Anxiety
  • Purple — Surprise / Awakening

Station-by-Station Notes

  • Finch → Sheppard — anticipation, quiet nerves
  • Sheppard → Eglinton — heaviness, routine
  • Eglinton → Bloor — overwhelm, emotional noise
  • Bloor → Union — grounding, soft return to self

The Result

A soft, colour-based emotional map visualizing a daily TTC journey.
